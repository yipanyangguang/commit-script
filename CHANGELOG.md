# 更新日志

## [1.4.0] - 2025-12-26

### 新增
- **高级作者筛选**: 基本配置中的“统计对象”功能增强，支持同时配置“包含用户”和“排除用户”。
- **智能筛选**: 支持从别名配置中选择用户，也支持手动输入关键字进行模糊匹配。

## [1.3.0] - 2025-12-26

### 新增
- **代码变更查看**: 预览与导出页面新增“查看变更”功能，支持查看 Commit 的详细代码变动。
- **Diff 可视化**: 集成 `diff2html`，提供美观的 Diff 对比视图（支持语法高亮、并排对比）。
- **成员代码统计**: 数据概览新增“成员代码变动统计”图表，展示增加行数、减少行数及净增加行数。

### 优化
- **图表布局**: 优化数据概览中提交类型分布饼图的布局，解决图例拥挤问题。
- **弹窗体验**: 优化变更详情弹窗样式，支持长代码滚动查看，适配不同屏幕尺寸。

### 修复
- **警告修复**: 修复控制台关于 `antd` Space direction 和 `rc-collapse` 的警告。
- **类型修复**: 修复 Dashboard 组件中的 TypeScript 类型错误。

## [1.2.0] - 2025-12-25

### 新增
- **图片导出**: 仪表盘（Dashboard）新增“导出图片”功能，支持将当前的分析报表保存为 PNG 图片。
- **国际化**: 全面支持中文（zh-CN）界面，优化了日期选择器和组件的中文显示。

### 优化
- **UI 改进**: 优化了贡献热力图（Heatmap）的布局和样式，使其更加美观易读。
- **窗口状态**: 恢复了窗口状态记忆功能，应用重启后可保持上次的窗口大小和位置。

### 修复
- **权限修复**: 修复了文件写入权限问题，确保图片导出功能在 macOS 上正常工作。
- **代码修复**: 修复了应用入口文件的语法结构问题。

## [1.1.0] - 2025-12-25

### 新增
- **深度代码透视**:
  - 新增代码行数统计（新增/删除）。
  - 新增贡献热力图（Contribution Heatmap），直观展示最近 6 个月的活跃度。
  - 新增工作时段分析，展示提交的时间分布。
  - 新增代码变动趋势图，展示每日代码增删情况。

### 修复
- **状态检查**: 修复了并发检查仓库状态时可能导致的数据不一致问题。
- **构建修复**: 修复了 TypeScript 类型定义错误。

## [1.0.0] - 2025-12-24

### 新增
- **数据概览 (Dashboard)**: 新增可视化仪表盘，展示提交类型分布、每日提交趋势及活跃作者排行。
- **快速跳转**: 预览列表支持直接点击跳转到远程仓库的 Commit 页面。
- **别名支持**: 数据概览中的活跃作者排行现在支持显示配置的作者别名。

### 优化
- **自动补充 Remote URL**: 在检查仓库状态时，会自动尝试获取并补充缺失的远程仓库地址。
- **UI 调整**: 修复了数据概览页面的水平滚动条问题。
- **构建修复**: 修复了 Rust 后端编译警告及权限配置问题。

## [0.6.0] - 2025-12-24

### 新增
- 新增“别名配置”功能，支持为 Git 作者设置显示别名。
- 项目配置页新增“上次检查时间”显示（如：几分钟前）。

### 优化
- 项目配置页的分组折叠状态支持持久化保存。
- 调整了 Tab 页签顺序，将“别名配置”移至“说明”之前。

## [0.5.1] - 2025-12-23

### 新增
- 预览列表支持点击“更多”查看完整 Commit 详情。


## [0.5.0] - 2025-12-22

### 新增
- 项目配置页显示仓库上次检查更新的时间。

### 变更
- 项目配置页“获取状态”按钮改为分组控制，支持单独获取某个分组的状态。
- 预览与导出表格新增“序号”列。

### 优化
- 优化获取状态时的交互体验，增加当前正在检查的项目提示。

## [0.4.0] - 2025-12-18

### 变更
- 移除自动获取远端状态的逻辑，改为手动控制。
- 项目配置页新增“获取状态”按钮，支持手动检查更新。
- 合并“时间配置”和“作者配置”为“基本配置”页面。

## [0.3.0] - 2025-12-18

### 新增
- 新增仓库更新预检功能 (git fetch --dry-run)，智能识别远端更新。
- 项目配置页新增仓库状态显示（⚠️ 远端有更新 / ✅ 已是最新）。
- 支持按分组一键更新所有有变更的仓库。

### 优化
- 优化分析流程，仅对有更新的仓库执行 fetch 操作，大幅缩短等待时间。
- 优化 UI 交互，切换到配置页时自动检查更新。

## [0.2.0] - 2025-12-17

### 新增
- 支持在分析前自动拉取远端代码 (git fetch --all)。
- 添加了更新日志显示功能。

### 优化
- 优化了分析过程中的加载提示。

## [0.1.0] - 2025-12-01

### 初始版本
- 支持多仓库配置。
- 支持按时间范围和作者筛选提交记录。
- 支持导出周报/日报格式文本。
